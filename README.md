# Tourism-Spot-Prediction-Android-App


\f0\fs24 \cf0 To run this Android Application, you need to have the following dependencies:\
\
You need to run the integration script which includes the API writing, you need to keep that running in the background.\
Then, you need to set up ngork in your system which will enable hosting between the network and the local server.\
Then you need to have an Android phone with a minimum SDK version of 24.\
\
Then you need to input all the required fields on the first page of the application such as Gender , budget etc\
Then on submission of these attributes, on the next page of the application you can see your top 5 tourist destinations.\
}
